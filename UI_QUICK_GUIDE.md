## 🎨 定制颜色

### 改变主色 (从蓝色改为其他颜色)

打开 `public/css/style.css`，找到第 8-9 行:

```css
:root {
    --primary-color: #2563eb;      /* 改这里 */
    --primary-dark: #1e40af;       /* 改这里 */
    --primary-light: #3b82f6;      /* 改这里 */
```

常用颜色:
- 绿色: `#10b981` (成功)
- 红色: `#ef4444` (警告)
- 紫色: `#8b5cf6` (优雅)
- 黄色: `#f59e0b` (提示)
