**功能特性**:
- ✅ 分页显示（每页10/25/50条）

**表格展示**:
```
┌─────┬─────────────┬──────────┬────────┬─────────────┐

└─────┴─────────────┴──────────┴────────┴─────────────┘
```

**操作**:
- 🔍 查看详情（完整Cookie信息）

### 3️⃣ Cookie导入 (Cookie Import)

**功能特性**:
- ✅ 文本输入模式
- ✅ 文件上传模式
- ✅ 拖拽上传支持
- ✅ 多格式支持（txt/csv/json）
- ✅ 实时进度显示
- ✅ 详细的导入反馈

**支持格式**:
```
基础格式:  IP cookie
扩展格式:  IP cookie validUntil

示例:
192.168.1.1 sessionid=xxx; path=/; secure
192.168.1.2 sessionid=yyy 2024-02-20T10:00:00Z
```

**导入流程**:
```
输入数据 → 格式验证 → 上传API → 进度显示 → 结果反馈
```

---

### 4️⃣ 批量操作 (Batch Operations)

**功能特性**:
- ✅ 批量ID输入
- ✅ 四种操作类型
- ✅ 操作预览
- ✅ 执行结果显示
- ✅ 二次确认保护

**支持操作**:

| 操作 | 图标 | 说明 |
|------|------|------|
| 验证 | ✓ | 批量验证选中Cookie的有效性 |
| 黑名单 | 🚫 | 批量添加到黑名单 |
| 释放 | 🔓 | 释放正在使用的Cookie |
| 删除 | 🗑️ | 批量删除Cookie |

**ID输入格式**:
```
方式1: 1,2,3,4,5
方式2: 1
      2
      3
方式3: 1, 2, 3 (混合)
```
