## ✨ 核心功能模块

### 1. 仪表板 (Dashboard) 📊

**功能**:
- 6个统计卡片（总数、可用、使用中、失效、黑名单、平均使用次数）
- 状态分布甜甜圈图
- 使用频率柱状图
- 自动30秒刷新

**技术**:
- Chart.js 3.9.1 数据可视化
- 响应式卡片布局
- 实时数据绑定

### 2. Cookie列表 📋

**功能**:
- 分页显示Cookie（每页10/25/50条）
- IP地址搜索（模糊匹配）
- 按状态过滤（可用/使用中/失效/黑名单）
- 单条快速操作（查看详情/验证/黑名单）
- 批量选择

**API调用**:
- `GET /api/cookies?page=&pageSize=&status=&ip=`
- `GET /api/cookies/:id`
- `POST /api/cookies/:id/validate`
- `POST /api/cookies/:id/blacklist`

### 3. Cookie导入 📥

**功能**:
- 文本输入方式
- 文件上传方式（支持txt/csv/json）
- 拖拽上传
- 导入进度显示
- 实时反馈（成功/失败数）

**格式支持**:
```
IP cookie [validUntil]
```

**API调用**:
- `POST /api/cookies/import`

### 4. 批量操作 🔄

**功能**:
- 验证选中Cookie ✓
- 添加黑名单 🚫
- 释放正在使用的Cookie 🔓
- 删除Cookie 🗑️
- 操作预览
- 执行结果显示

**ID输入格式支持**:
```
1,2,3        逗号分隔
1
2
3            每行一个
```

**API调用**:
- `POST /api/cookies/validate/batch`
- `POST /api/cookies/blacklist/batch`
- `POST /api/cookies/:id/release`

### 5. 系统设置 ⚙️

**功能**:
- 服务器信息显示
- CSV导出
- JSON导出
- 清理失效Cookie
- 清空黑名单

**API调用**:
- `GET /api/statistics`
- `GET /api/cookies` (导出)

## 🚀 使用指南

### 快速启动

```bash
# 1. 启动服务
npm run dev

# 2. 访问管理页面
http://localhost:3000

# 3. 开始使用
```

### 主要工作流

**导入Cookie工作流**:
```
1. 切换到"导入Cookie"标签
2. 选择文本输入或文件上传
3. 粘贴/上传Cookie数据
4. 点击"开始导入"
5. 查看导入结果
6. 返回Dashboard查看统计
```

**管理Cookie工作流**:
```
1. 切换到"Cookie列表"标签
2. 使用搜索/过滤找到目标Cookie
3. 单条操作（验证/黑名单/查看详情）
4. 或批量选择进行批量操作
5. 实时查看结果
```
