# 快速启动指南 - 管理页面

## 🚀 一键启动

### 步骤 1: 启动服务

```bash
# 进入项目目录
cd xiaohongshu-cookie-pool

# 如果还没安装依赖
npm install

# 启动开发服务
npm run dev
```

### 步骤 2: 打开管理页面

在浏览器中访问：

```
http://localhost:3000
```

### 步骤 3: 完成！

您应该看到一个美观的管理界面，包含：

- 📊 仪表板（实时统计）
- 📋 Cookie列表（搜索/过滤）
- 📥 导入Cookie（文本/文件）
- 🔄 批量操作（验证/黑名单）
- ⚙️ 系统设置（导出/清理）

## 🎯 常见操作

### 导入Cookie

1. 点击左侧菜单 "导入Cookie"
2. 选择输入方式（文本或文件）
3. 粘贴或上传Cookie数据
4. 点击 "开始导入"

### 查看统计

1. 点击左侧菜单 "仪表板"
2. 查看6个统计卡片
3. 查看状态分布和使用频率图表

### 管理Cookie

1. 点击左侧菜单 "Cookie列表"
2. 使用搜索和过滤功能
3. 点击操作按钮进行单条操作
4. 或勾选后进行批量操作

## ⚠️ 故障排除

### 页面无法加载？

```bash
# 1. 检查服务是否运行
npm run dev

# 2. 验证port 3000是否被占用
netstat -an | grep 3000

# 3. 如果被占用，修改环境变量
PORT=3001 npm run dev
```

### API调用失败？

检查浏览器开发者工具 (F12)：
- Network 标签查看API调用状态
- Console 标签查看错误信息

### 数据不显示？

```bash
# 1. 检查数据库连接
# 2. 确认MySQL服务运行
# 3. 查看后端日志
tail -f logs/app.log
```
