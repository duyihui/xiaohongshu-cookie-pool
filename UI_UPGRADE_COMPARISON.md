## 🌀 动画新增

### fadeIn 淡入
```
内容切换时使用
从 opacity: 0 → 1
duration: 300ms
用于标签切换
```

### slideInRight 滑入
```
通知提示使用
从 translateX(100px) → 0
opacity: 0 → 1
duration: 300ms
自右侧飞入
```

### slideUp 滑上
```
模态框打开时
从 translateY(20px) → 0
opacity: 0 → 1
duration: 300ms
自下方浮上
```

### pulse 脉冲
```
在线状态指示器
box-shadow 脉冲扩展
duration: 2s
循环脉冲
```
