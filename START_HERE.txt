🎯 核心API (最常用的4个)

  1️⃣  导入Cookie
      POST /api/cookies/import
      
      请求：
      {
        "cookies": [
          {"ip":"192.168.1.1","cookie":"sessionid=abc123"}
        ]
      }

  2️⃣  获取随机Cookie
      GET /api/cookies/random
      
      响应：
      {
        "id": 1,
        "ip": "192.168.1.1",
        "cookie": "sessionid=abc123",
        "useCount": 5
      }

  3️⃣  释放Cookie
      POST /api/cookies/1/release

  4️⃣  获取统计
      GET /api/statistics
      
      响应：
      {
        "total": 100,
        "available": 80,
        "using": 5,
        "invalid": 10,
        "blacklist": 5
      }

💻 完整的工作流

  1. 导入 Cookie
     POST /api/cookies/import
     
  2. 获取可用 Cookie
     GET /api/cookies/random
     返回: {id, ip, cookie}
     
  3. 使用 Cookie 爬取
     设置代理IP
     使用返回的Cookie
     
  4. 任务完成释放
     POST /api/cookies/{id}/release
